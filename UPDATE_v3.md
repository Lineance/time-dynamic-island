# 灵动岛 v3.0 - 更新说明

## ✅ 已修复的问题

### 1. 文字显示问题
- **修复前**: 看不见时间文字
- **修复后**: 使用 Tahoma 字体 + ClearType 渲染，文字清晰可见
- 文字颜色：白色 RGB(255,255,255)（深色主题）
- 字体大小：14 号加粗

### 2. 鼠标悬停问题
- **修复前**: 鼠标挪上去后左移而不是放大
- **修复后**: 正确在原位置放大（120x25 → 145x30）
- 使用新的 `UpdateWindowPosition()` 函数确保居中

### 3. 右键菜单问题
- **修复前**: 右键点击没反应
- **修复后**: 右键点击正常弹出设置对话框

### 4. 文件命名
- **修改**: 删除所有文件名中的"iphone"字样
- `main.cpp` → 使用标准命名

## 🆕 新增功能

### 自动隐藏功能

#### 设置选项
在右键设置对话框中新增：
- **Auto-hide (show on hover)** - 自动隐藏（鼠标悬停时显示）

#### 功能说明
- **默认**: 不隐藏（正常显示）
- **启用后**: 灵动岛隐藏在屏幕上方外面（Y=-50）
- **鼠标移动到屏幕顶部**: 灵动岛自动显示
- **鼠标移开**: 灵动岛自动隐藏回屏幕外

#### 使用场景
- 观看全屏视频时不打扰
- 需要时鼠标一挪到顶部就能看到
- 保持桌面极简

## 🔧 技术改进

### 1. 渲染优化
- 使用 `BitBlt` 双缓冲渲染
- 负字体高度确保字符大小精确
- ClearType 字体渲染

### 2. 位置管理
- 新增 `g_currentX`, `g_currentY` 位置跟踪
- 新增 `g_hiddenY` 隐藏位置
- `UpdateWindowPosition()` 统一处理位置

### 3. 自动隐藏逻辑
```cpp
if (g_autoHide) {
    y = g_isHovering ? visibleY : hiddenY;
}
```

## 🎯 使用说明

### 基本操作
1. **左键点击**: 打开系统时间设置
2. **右键点击**: 打开灵动岛设置
3. **鼠标悬停**: 灵动岛放大
4. **自动隐藏**: 启用后鼠标移到屏幕顶部显示

### 设置选项
- **Time Format**: 24 小时制 / 12 小时制
- **Display Options**:
  - Show Date: 显示日期
  - Show Year: 显示年份
  - Show Seconds: 显示秒钟
  - Auto-hide: 自动隐藏
- **Theme**: 深色 / 浅色 / 跟随系统
