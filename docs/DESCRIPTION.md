# 灵动岛 (Dynamic Island)

## 📝 项目描述 / Project Description

**中文：**

灵动岛是一个优雅的 Windows 桌面时间显示工具，采用现代胶囊设计风格。

**开发动机：** 我喜欢隐藏任务栏并使用任务栏透明化工具，但在某些特定情况下无法看见任务栏的时间。为了解决这个痛点，我开发了灵动岛——一个独立于任务栏的时间显示工具，始终悬浮在屏幕顶部，让你随时都能看到时间。

**主要特性：**
- 🕐 实时时间显示，每秒自动更新
- 🎨 现代胶囊形状，精致美观
- 📅 可选显示日期、年份、秒钟
- 🎯 可自定义位置（左/中/右）、尺寸、主题
- 🖱️ 鼠标悬停放大，交互流畅
- 💾 极低内存占用（约 2MB）
- 🔒 纯外部程序，不涉及游戏内存修改

---

**English:**

Dynamic Island is an elegant Windows desktop time display tool featuring a modern capsule design.

**Motivation:** I prefer to hide the taskbar and use taskbar transparency tools, but in certain situations, I couldn't see the taskbar clock. To solve this pain point, I created Dynamic Island - an independent time display tool that always floats at the top of the screen, allowing you to check the time anytime.

**Key Features:**
- 🕐 Real-time clock with auto-update every second
- 🎨 Modern capsule shape, elegant and beautiful
- 📅 Optional date, year, and seconds display
- 🎯 Customizable position (left/center/right), size, and theme
- 🖱️ Smooth hover animation for interactive experience
- 💾 Ultra-low memory footprint (~2MB)
- 🔒 Pure external program, no game memory modification
